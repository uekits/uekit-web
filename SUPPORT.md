# 支持说明

## 获取帮助

- 使用问题、可复现缺陷：通过相应 GitHub Issue 模板提交。
- 功能建议：使用 Feature request 模板，并说明通用开源场景和替代方案。
- 安全漏洞：不要创建公开 Issue，请遵循 [`SECURITY.md`](./SECURITY.md)。

提问前请先检索现有 Issue 和文档。问题中应提供 UEKit CLI 版本、Node 与包管理器版本、消费项目技术栈、最小复现和完整错误信息；请删除 Token、密钥、业务数据和内部地址。

社区支持按维护者可用时间提供，不承诺 SLA。商业项目的紧急交付、私有定制和与 UEKit 无关的业务代码排查不属于公共 Issue 的保证范围。
